BlossomMovie is a SwiftUI-based iOS application for browsing and exploring movies.  
The app fetches data from a movie API and displays trending, popular, and detailed movie information.

---

## Features

- SwiftUI-based UI  
- MVVM architecture  
- Movie search  
- Movie details screen  
- YouTube trailer support  
- Async/Await networking  

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/miss-linetta/BlossomMovie.git  
cd BlossomMovie
```

### 2. Configure API keys

Before running the project, you must configure your API keys.

Open: 
```bash 
BlossomMovie/APIConfig.example.json 
``` 

Rename the file to:  
```bash
BlossomMovie/APIConfig.json  
```

Replace the placeholder values with your own API keys.
